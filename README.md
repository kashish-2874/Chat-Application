# Chat Application

## Introduction
This project is a full-stack real-time chat application designed to enable seamless communication between users. It is built using modern web technologies and follows a client-server architecture. The application focuses on secure authentication, efficient data handling, and real-time message delivery.

## Objective
The main objective of this project is to implement a scalable and secure chat system that demonstrates real-time communication using WebSockets along with a robust backend and responsive frontend.

## Key Features
- **User registration and login functionality**
- **Secure authentication using JWT or HTTP-only cookies**
- **Real-time messaging using Socket.IO**
- **One-to-one private chat system**
- **Protected routes for authorized access**
- **Persistent data storage using MongoDB**
- **Responsive and user-friendly interface**

## Tech Stack

### Frontend
- **React.js**
- **Axios**
- **CSS / Tailwind CSS**

### Backend
- **Node.js**
- **Express.js**

### Database
- **MongoDB**

### Real-Time Communication
- **Socket.IO**

### Authentication
- **JSON Web Tokens (JWT) / Cookie-based authentication**

## System Architecture
The application follows a typical MERN stack architecture:
- The frontend handles UI and user interaction  
- The backend manages APIs and business logic  
- MongoDB stores user and message data  
- Socket.IO enables real-time communication
